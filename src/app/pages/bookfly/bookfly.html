BookFly

Plataforma social para leitores compartilharem avaliações, criarem comunidades e descobrirem novos livros através de interesses literários em comum.

sobre o projeto.

Problema

Leitores muitas vezes têm dificuldade para encontrar pessoas com gostos parecidos e acompanhar suas leituras em um ambiente voltado para livros.

Solução

O BookFly centraliza avaliações, comunidades, estantes e recomendações, criando uma experiência social focada no universo literário.

Objetivo

Construir uma aplicação web completa, aplicando conceitos de frontend, backend, banco de dados, integração externa e arquitetura em camadas.

funcionalidades.

  • Avaliação de livros
  • Criação de comunidades literárias
  • Comentários e interações entre usuários
  • Estantes personalizadas
  • Metas de leitura
  • Integração com a Google Books API

arquitetura.

O backend foi desenvolvido com ASP.NET Core, utilizando PostgreSQL como banco de dados e NHibernate para o mapeamento objeto-relacional. A aplicação segue uma organização em camadas, separando responsabilidades entre API, aplicação, domínio e infraestrutura.

No frontend, a interface foi construída com Angular e TypeScript, consumindo os endpoints da API e exibindo os dados relacionados a livros, comunidades, avaliações e usuários.

Fiquei responsável pela implementação do backend e pela integração com a Google Books API.
