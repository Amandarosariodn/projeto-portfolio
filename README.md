## Running the Project

Clone the repository:

```bash
git clone https://github.com/Amandarosariodn/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
ng serve
```

Open:

```txt
http://localhost:4200
```
